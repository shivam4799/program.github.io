Fibonacci Series

The Fibonacci sequence is a series of numbers where a number is the sum of previous two numbers. Starting with 0 and 1, the sequence goes 0, 1, 1, 2, 3, 5, 8, 13, 21, and so on.



Check palindrome number algorithm

  1. First initialize two number of the Fibonacci sequence is 0 & 1.
  2. Next term is the sum of pervious two terms.
  3. Print that number.

Program in C :


  #include <stdio.h>
  #include <string.h>
  int main()
  {
    int n, first = 0, second = 1, next, c;
    
    printf("Enter the number of terms\n");
    scanf("%d", &n);
    
    printf("First %d terms of Fibonacci series are:\n", n);
    
    for (c = 0; c < n; c++)
    {
      if (c <= 1)
        next = c;
      else
      {
        next = first + second;
        first = second;
        second = next;
      }
      printf("%d\n", next);
    }
    
    return 0;
  }

Program in JAVA :


    public class Fibonacci {

      public static void main(String[] args) {
  
          int n = 10, t1 = 0, t2 = 1;
          System.out.print("First " + n + " terms: ");
  
          for (int i = 1; i <= n; ++i)
          {
              System.out.print(t1 + " + ");
  
              int sum = t1 + t2;
              t1 = t2;
              t2 = sum;
          }
      }
  }
